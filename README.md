#### Installation

1. Clone the repository:
```bash
git clone https://github.com/SuperteamDAO/earn.git
```

2. Navigate to the project directory:
```bash
cd earn
```

3. Install the dependencies: 
```bash
yarn install
```

4. Run the development server
```bash
yarn dev
```
